# Windy Premium Bypass

Инструмент для расширения стандартного функционала и разблокировки премиальных метеорологических данных на платформе Windy.com.

---

## Методы установки

### 1. Букмарклет
*Наиболее быстрый способ, не требующий установки расширений.*

1. Откройте [страницу проекта](https://cruitac345.github.io/WindyUnlocker/).
2. Перетащите кнопку **Windy Bypass** на панель закладок браузера.
3. Перейдите на сайт [windy.com](https://www.windy.com).
4. Нажмите на сохраненную закладку после загрузки страницы.

### 2. Консоль разработчика
*Для разового запуска без сохранения.*

1. Откройте [windy.com](https://www.windy.com).
2. Нажмите `F12` и перейдите во вкладку **Console**.
3. Вставьте основной код со [страницы проекта](https://cruitac345.github.io/WindyUnlocker/).
4. Нажмите `Enter`.

### 3. Tampermonkey
*Автоматическая активация при каждом посещении.*

1. Установите расширение [Tampermonkey](https://www.tampermonkey.net/).
2. Скачайте `userscript` со [страницы проекта](https://cruitac345.github.io/WindyUnlocker/).
3. Подтвердите установку в окне расширения.

## Основные возможности

| Функция | Описание |
| :--- | :--- |
| **Прогноз на 10 дней** | Снятие стандартного ограничения в 3 дня. |
| **Почасовой прогноз** | Детализация данных с шагом в 1 час. |
| **Премиум слои** | Доступ к архивам спутников и картам высокого разрешения. |
| **Расширенные данные** | Доступ к дополнительным метеорологическим моделям. |
| **Отключение рекламы** | Удаление маркетинговых блоков и предложений подписки. |
